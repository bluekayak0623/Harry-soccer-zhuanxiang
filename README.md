# Harry Soccer Training

足球专项训练动态 Web App，适合部署到 Vercel。

## 功能

- 16:9 横向足球场训练界面
- 1-8 号区域随机闪烁且一轮内不重复
- 开始倒计时、单元休息倒计时、组间休息倒计时语音提示
- 暂停、继续、重置控制
- 右侧显示训练组、单元、闪烁次数和总进度

## 本地运行

直接双击 `index.html`，或用任意静态服务器打开本目录。

## Vercel

此项目是静态站点，Vercel 导入仓库后无需构建命令即可部署。

部署步骤：

1. 打开 https://vercel.com/new/import
2. 在 Import Git Repository 中选择 GitHub
3. 搜索 `Harry-soccer-zhuanxiang`
4. 点击 Import
5. 保持默认配置，点击 Deploy
